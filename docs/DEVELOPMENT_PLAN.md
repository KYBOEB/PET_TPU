# DEVELOPMENT PLAN — Питомец ТПУ 🐶

**Срок:** 4 недели (1 месяц).

## Чек-лист серверных требований (задача 4.7)

Перед деплоем backend-команда должна получить от администратора сервера:

- [ ] IP-адрес / hostname сервера
- [ ] SSH-доступ (логин + способ аутентификации)
- [ ] Версия ОС (`cat /etc/os-release`)
- [ ] Наличие Python 3.11+ (`python3 --version`)
- [ ] Наличие Docker (`docker --version`) — опционально
- [ ] Наличие GPU (`nvidia-smi`) — для локального LLM
- [ ] Свободное место на диске (`df -h`) — нужно минимум 1 ГБ (без LLM) / 10 ГБ (с локальным LLM)
- [ ] Открытые порты (80, 443, 8000)
- [ ] Наличие sudo-прав
- [ ] Доменное имя (если планируется)

**Бэкенд-команда управляет сервером самостоятельно.**
